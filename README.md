scala_sample
============

sample for Scala.

Number place solver
------------
* directory
number_place
* how to run
scala nplace.scala (number sequence. 1-9 for initially placed numbers and 0 for empty squares)

Calculator
------------
* directory  
calculator
* how to build  
scalac Arith.scala  
scalac Calculator.scala
* how to run  
scala Calculator "formula"  
ex. scala "(1 + 0.5) * 3"
* supported oparators  
+, -, * and /

Actor sample
------------
* directory  
actor
* how to run  
scala actor.scala
